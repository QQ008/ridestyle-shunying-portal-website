技术栈 API文档 开发指南

技术栈

瞬影项目采用现代化的技术栈，注重性能、可扩展性与开发效率

前端技术

  • Vue 3 渐进式JavaScript框架，用于构建用户界面
  • TypeScript 提供类型系统，增强代码可靠性与开发体验
  • Vite 下一代前端构建工具，提供极速的开发体验
  • TailwindCSS 实用优先的CSS框架，用于快速构建现代化界面
  • Pinia Vue的状态管理库，简单直观且类型安全

后端技术

  • Nest.js 进步的Node.js框架，用于构建高效、可靠的服务器端应用
  • TypeScript 提供强类型系统，确保API接口的可靠性
  • PostgreSQL 先进的开源关系型数据库，支持复杂查询和地理信息
  • Redis 内存数据结构存储，用于缓存和会话管理
  • MinIO 高性能对象存储服务，用于存储和管理照片资源

AI & 图像处理

  • Python 主要用于AI模型训练与图像处理服务
  • YOLO v11 用于实时目标检测，识别骑行者与自行车
  • OpenCV 开源计算机视觉库，用于图像处理与分析
  • TensorFlow 用于构建和训练骑行者识别模型
  • FastAPI 现代、高性能的Python API框架，用于图像处理服务

DevOps & 基础设施

  • Docker 容器化技术，确保开发与生产环境一致性
  • Kubernetes 容器编排系统，用于管理分布式应用
  • GitHub Actions CI/CD流程自动化，确保代码质量与快速部署
  • Prometheus & Grafana 监控系统，用于跟踪系统性能与健康状态
  • 阿里云 云服务提供商，提供稳定、可扩展的基础设施

系统架构

瞬影采用微服务架构，各组件独立部署，提高系统可靠性与可扩展性

前端应用

包括用户端移动应用和摄影师管理后台，采用Vue.js构建，通过API网关与后端服务交互

API网关

统一的API入口，负责请求路由、负载均衡、认证授权、限流等功能

核心服务

包括用户服务、照片管理服务、交易服务等，每个服务独立部署，通过消息队列异步通信

AI服务

提供图像识别、废片筛选、骑行者匹配等AI能力，独立扩展，通过异步任务处理大规模图像

存储层

包括关系型数据库、对象存储和缓存系统，确保数据安全和高效访问

开发指南

快速上手瞬影项目开发

1

环境准备

首先确保您的开发环境已安装以下工具：

  • Node.js v16+
  • Docker & Docker Compose
  • Git
  • VSCode (推荐)
  • Python 3.9+ (用于AI服务开发)
2

获取代码

git clone https://github.com/momentshot/momentshot.git
cd momentshot
npm install
3

启动开发环境

# 启动依赖服务 (PostgreSQL, Redis, MinIO等)
docker-compose up -d

# 启动前端开发服务器
npm run dev:web

# 启动后端服务
npm run dev:api
4

代码提交

我们使用GitFlow工作流，请按以下步骤提交代码：

  1. 从main分支创建功能分支: git checkout -b feature/your-feature
  2. 开发完成后，提交代码: git commit -m "feat: your feature description"
  3. 推送到远程仓库: git push origin feature/your-feature
  4. 在GitHub上创建Pull Request，等待代码审核
